# Instalacja ze źródeł

* Wymagania

Zainstalowane Node.js https://nodejs.org

* Pobieranie

git clone https://github.com/diabelb/OpenFlix.git

* Instalacja

cd OpenFlix

npm install

* Uruchomienie

npm start
